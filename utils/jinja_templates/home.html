Fluffy Hero Maker

This internal website allows you to access all the tools you want from the Fluffy Hero Maker suite.

Available Tools

DnD 5:

  • Generate a random character, you can change "Someone" into any name you want for the character
  • Get a random spell
  • Get a random monster
  • Get the list of all monsters
  • Get the list of all spells

Classic Traveller:

  • Generate a random character, you can change Sara into another name
